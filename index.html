Bienvenido

Práctico es un proyecto de Software Libre publicado bajo licencia GNU GPL v2.0 para la creación de aplicaciones web de una manera completamente visual y rápida. Sin conocimientos de programación.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

Construya su aplicación web:

  1. Descargue la última versión estable, descomprima sobre su servidor web y abra su navegador en la dirección correspondiente.
  2. Cree las primeras tablas de datos
  3. Diseñe los formularios para ingreso de información
  4. Genere reportes automáticos a partir de las tablas
  5. Cree opciones de menú para accesar a cada formulario o informe diseñado
  6. Defina los usuarios y permisos para cada uno.
